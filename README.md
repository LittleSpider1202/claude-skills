# Claude Skills & Tools

Claude Code 的 skills 和 tools 合集。

## 安装

```bash
# 将 skills 符号链接到全局目录
# Linux/macOS
ln -s /path/to/claude-skills/skills/* ~/.claude/skills/

# Windows (管理员 PowerShell)
# 逐个链接或直接复制
```

## Skills

| coding-standards | 开发工作流 | 通用编码规范（TypeScript/JavaScript/React/Node.js） |
